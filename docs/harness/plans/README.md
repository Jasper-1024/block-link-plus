# Harness Plans

This directory tracks cross-task harness improvements. It does not replace
per-task run artifacts under `docs/harness/runs/<archive-key>/`.

Use this area for work that affects more than one Plane item:

- active harness improvement plans
- completed improvement summaries
- known technical debt
- recurring doc-gardening findings

Keep plans concise and link to canonical run artifacts when a specific issue
provided the evidence.
