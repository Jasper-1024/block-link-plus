# Runtime Evidence

This directory contains current, reusable runtime knowledge for agent runs.
Historical debug logs and older design discussions belong under
`archive/internal-docs/`.

- [isolated-obsidian-cdp.md](isolated-obsidian-cdp.md): launch and verify a
  disposable Obsidian Desktop runtime.
- [cdp-script-inventory.md](cdp-script-inventory.md): choose reusable CDP
  snippets under `scripts/cdp-snippets/`.
- [fixtures/](fixtures/): Markdown inputs used by CDP runtime commands.

Use [../harness/guides/cdp-runtime.md](../harness/guides/cdp-runtime.md) for
the stage gate. Use this directory for the detailed runtime references behind
that gate.
